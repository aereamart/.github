# Aereamart 🛒

Aereamart is a modern ERP and e-commerce platform that brings retail operations, inventory management, procurement, and financial workflows together in one place. We help Sri Lankan businesses streamline their daily operations — from point-of-sale and customer engagement to supply chain and accounting — so they can focus on growth.

Built for retailers, wholesalers, and distributors who need a reliable, all-in-one solution to run their business better.
